# 键盘说明

## 1. 说明
本文件用于详细说明 `8x8` 键盘的每一个按键。  
主键位布局来源于 [PRD.md](./PRD.md) 的键位规划章节，本文件只负责逐键定义，避免在 `PRD` 中重复展开。

## 2. 主层键位总表

| 行\列 | C1 | C2 | C3 | C4 | C5 | C6 | C7 | C8 |
|---|---|---|---|---|---|---|---|---|
| R1 | ON | OFF | MENU | MODE | HIST | FILE | UP | DEL |
| R2 | SHIFT | ALPHA | LEFT | OK | RIGHT | ( | ) | BACK |
| R3 | 7 | 8 | 9 | / | sin | cos | tan | π |
| R4 | 4 | 5 | 6 | * | ln | log | x^y | e |
| R5 | 1 | 2 | 3 | - | sqrt | x^2 | 1/x | % |
| R6 | 0 | . | +/- | + | EXP | ENG | Ans | = |
| R7 | ENTER | SWAP | DROP | DUP | ROLL↑ | ROLL↓ | STO | RCL |
| R8 | MC | MR | M+ | M- | DEG | RAD | BIN | HEX |

## 3. 逐键说明

### 3.1 R1 行

| 键位 | 主功能 | `SHIFT` 副功能 | 说明 |
|---|---|---|---|
| R1C1 `ON` | 开机 / 唤醒 | 自检页面 | 打开设备，长按可进入启动自检 |
| R1C2 `OFF` | 关机 / 待机 | 强制保存后关机 | 退出当前会话前尝试关闭日志文件 |
| R1C3 `MENU` | 打开系统菜单 | 快速系统设置 | 进入模式、显示、存储等设置入口 |
| R1C4 `MODE` | 切换工作模式 | 模式列表 | 在科学/程序员/历史等模式间切换 |
| R1C5 `HIST` | 打开历史浏览 | 跳到最新记录 | 浏览本次或过往会话记录 |
| R1C6 `FILE` | 文件管理 | 日志导出/删除入口 | 浏览 `1.txt / 2.txt / 3.txt` 等文件 |
| R1C7 `UP` | 上移光标/上翻 | 快速上翻 | 用于菜单、历史、文件列表导航 |
| R1C8 `DEL` | 删除当前输入字符 | 清空当前输入行 | 删除数字、符号或编辑中的表达式 |

| 键位 | 主功能 | `SHIFT` / `ALPHA` 副功能 | 说明 |
|---|---|---|---|

| R2C2 `ALPHA` | 启用字符层 | 锁定 `ALPHA` | 在文件名、程序员模式等场景输入字符 |

| R2C4 `OK` | 确认/执行 | 快速确认 | 菜单确认、文件确认、结果回填确认 |
| R2C5 `RIGHT` | 右移光标 | 行尾 | 用于编辑与浏览界面 |
| R2C6 `(` | 左括号 | 列表左翻页 | 用于表达式回显或代数兼容输入 |
| R2C7 `)` | 右括号 | 列表右翻页 | 与左括号配对使用 |
| R2C8 `BACK` | 返回上一级 | 返回主界面 | 退出菜单、历史或文件界面 |

## 4. 分区设计思路
- 顶部两行主要负责系统导航、菜单、历史、文件和编辑。
- 中间四行主要负责高频数值输入与科学函数。
- 底部两行主要负责 `RPN` 栈操作、寄存器与模式切换。
- 这样布局的目标是把最高频按键集中在中部和右手易达区域。

## 5. 说明边界
- 本文档给出的是推荐键位定义，不是最终丝印锁定版。
- 若后续发现某些功能优先级变化，可以只改本文件与 `PRD` 中的总表，不必把详细说明散落到多个文档。
