# IO 接线说明

## 1. 设计原则
- 主控默认按 `ATmega328P / Arduino Nano(或 Uno)` 兼容引脚定义
- 避开 `D0(TX)` 与 `D1(RX)`，保留下载和调试能力
- `TF` 卡使用标准 `SPI`
- `LCD2004` 使用标准 `4-bit` 并口
- 键盘矩阵通过 `74HC595 + 74HC165` 扩展

## 2. 推荐引脚分配

### 2.1 LCD2004

| LCD2004 引脚/信号 | 连接到 ATmega328P | 说明 |
|---|---|---|
| RS | A0 | 指令/数据选择 |
| E | A1 | 使能信号 |
| D4 | A2 | 4 位数据线 |
| D5 | A3 | 4 位数据线 |
| D6 | A4 | 4 位数据线 |
| D7 | A5 | 4 位数据线 |
| RW | GND | 固定写模式 |
| VSS | GND | 地 |
| VDD | +5V | 电源 |
| VO | 电位器中点 | 对比度调节 |
| LED A/K | 模块规格决定 | 背光供电 |

说明：
- 该方案是标准 `4-bit` LCD 并口接法，可落地、稳定、资料多。
- 如果后续强制要求“只有 4 根控制线”，需重做硬件方案。

### 2.2 74HC595（键盘行输出）

| 74HC595 信号 | 连接到 ATmega328P | 说明 |
|---|---|---|
| SER | D2 | 串行数据输入 |
| SRCLK | D3 | 移位时钟 |
| RCLK | D4 | 锁存时钟 |
| OE | GND 或可选 PWM | 低有效，默认可直接使能 |
| SRCLR | +5V | 高有效保持正常工作 |
| Q0~Q7 | 键盘 R1~R8 | 行扫描输出 |
| VCC | +5V | 电源 |
| GND | GND | 地 |

### 2.3 74HC165（键盘列输入）

| 74HC165 信号 | 连接到 ATmega328P | 说明 |
|---|---|---|
| PL/SHLD | D5 | 并装/移位控制 |
| CP | D6 | 时钟输入 |
| Q7 | D7 | 串行输出到 MCU |
| CE | GND | 低有效，默认常使能 |
| DS | GND | 级联未用时拉低 |
| A~H | 键盘 C1~C8 | 列输入 |
| VCC | +5V | 电源 |
| GND | GND | 地 |

补充建议：
- `74HC165` 列输入应使用 `8` 路 `10k` 上拉电阻网络，不能悬空。
- 默认状态下各列保持高电平。
- 当 `74HC595` 将当前扫描行拉低、且对应按键按下时，该列会被拉低。
- 软件读取到低电平时判定该键按下。
- 本方案不建议列端做下拉，避免与“行低有效、按下读低”的扫描逻辑冲突。

### 2.4 TF 卡模块（SPI）

| TF 卡模块信号 | 连接到 ATmega328P | 说明 |
|---|---|---|
| CS | D10 | 片选 |
| MOSI | D11 | SPI 主输出 |
| MISO | D12 | SPI 主输入 |
| SCK | D13 | SPI 时钟 |
| VCC | 视模块而定 | 带电平转换模块可接 5V，否则应接 3.3V |
| GND | GND | 地 |

### 2.5 预留引脚

| 引脚 | 建议用途 |
|---|---|
| D8 | 蜂鸣器 / 状态灯 |
| D9 | SD 卡检测 / 故障指示 |
| D0/D1 | 保留，不使用 |

## 3. 键盘矩阵连接逻辑

### 3.1 行列定义
- `74HC595 Q0~Q7` 分别对应 `R1~R8`
- `74HC165 A~H` 分别对应 `C1~C8`

### 3.2 扫描流程
1. MCU 通过 `74HC595` 仅激活一行
2. 当前行若有按键闭合，对应列电平变化
3. MCU 通过 `74HC165` 读取 `8` 列状态
4. 轮流完成 `8` 行扫描，得到 `8x8` 键盘状态矩阵

### 3.3 电气建议
- 行输出与列输入之间不建议悬空
- 列端必须使用稳定上拉，推荐 `10k` 电阻网络
- 若对多键同时按下有更高要求，应评估是否增加二极管矩阵

## 4. 避开 TX/RX 的理由
- `D0/D1` 是下载、串口监视器、调试最常用接口
- 若占用 `TX/RX`，会影响烧录与调试稳定性
- 本方案把全部外设放在 `A0~A5`、`D2~D13`，保留 `D0/D1`

## 5. 推荐总表

| 功能模块 | 使用引脚 |
|---|---|
| LCD2004 | A0, A1, A2, A3, A4, A5 |
| 74HC595 | D2, D3, D4 |
| 74HC165 | D5, D6, D7 |
| 预留功能 | D8, D9 |
| TF 卡 SPI | D10, D11, D12, D13 |
| 串口保留 | D0, D1 |
